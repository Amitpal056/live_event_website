# Event Scraping and Display Project

A MERN stack application that scrapes events from public websites for Sydney, Australia, and displays them in a user-friendly web interface with an admin dashboard.

## Features

- Automatic event scraping from multiple sources
- Event listing with clean UI
- Email capture for ticket purchases
- Google OAuth authentication
- Admin dashboard with filters and import functionality

## Tech Stack

- **Backend:** Node.js, Express, MongoDB
- **Frontend:** React
- **Scraping:** Puppeteer
- **Authentication:** Google OAuth (Passport.js)

## Setup

1. Clone the repository
2. Install dependencies for backend and frontend
3. Set up MongoDB
4. Configure Google OAuth credentials
5. Run the scraper, backend, and frontend

## Usage

- Visit the website to browse events
- Log in with Google for admin dashboard
- Use dashboard to filter, view, and import events
